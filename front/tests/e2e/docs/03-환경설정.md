# 03. 환경설정 (환경변수)

값은 전부 `fixtures/test-data.ts` 가 읽는다. 코드에 흩뿌리지 말고 여기로 모은다.

---

## 필수

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `BASE_URL` | `http://localhost` | **대상 콘솔 주소.** 원격 개발 서버면 `http://cmig.dev.cscmzc.com` |
| `TEST_SOURCE_IP` | (없음) | **소스 서버 VM 공인 IP.** 비워 두면 수집이 실패하고 그 뒤가 전부 무너진다 |
| `TEST_SOURCE_PRIVATE_KEY` | (없음) | 소스 VM SSH 개인키 **내용**(경로 아님). `"$(cat e2e-source-key)"` |

## 자주 쓰는 것

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `E2E_INCLUDE_COSTLY` | (없음) | `1` 이면 실제 클라우드 자원을 만드는 테스트(`@costly`)도 실행 |
| `E2E_EVIDENCE` | (켜짐) | `0` 이면 성공 건의 화면·영상 저장을 끈다(CI 용량 절감) |
| `E2E_RUN_ID` | 자동 | 생성 리소스 이름 접미사. 보통 건드리지 않는다(아래 참조) |
| `TEST_USERNAME` / `TEST_PASSWORD` | `cmiguser` / `cmiguserPassword!` | 로그인 계정 |
| `TEST_SOURCE_SSH_USER` | `ubuntu` | 소스 VM SSH 계정 |

## 네임스페이스

| 변수 | 기본값 | 설명 |
|------|--------|------|
| `TEST_NS` | **`mig01`** | 마이그레이션 인프라가 들어가는 cb-tumblebug 네임스페이스 |

> ★ **콘솔과 같은 값이어야 한다.** 콘솔은 `src/shared/constants/namespace.ts` 의 `DEFAULT_NAMESPACE = 'mig01'` 을 쓴다.
> 예전에 여기만 `default` 로 두는 바람에, 워크로드가 늘 비어 보이고 원격명령(nginx 설치)이 대상 인프라를 못 찾았다.
> **콘솔이 보는 것과 테스트가 보는 것이 달라지면 안 된다.**

## 소스 서버에 옮길 소프트웨어가 있어야 한다

소프트웨어 마이그레이션 시나리오는 **"소스에서 돌던 소프트웨어가 타깃으로 옮겨졌는가"** 를 확인한다.
그러니 소스에 옮길 것이 *실제로 있어야* 한다. 빈 서버로 돌리면 마이그레이션이 옮길 게 없고,
테스트는 아무것도 증명하지 못한 채 통과해 버린다.

```bash
TEST_SOURCE_IP=1.2.3.4 TEST_SOURCE_KEY=~/.ssh/e2e-source.pem \
  ./scripts/prepare-source-server.sh
```

nginx를 올린다. 그러면 cm-honeybee 수집에 패키지(`nginx`)와 바이너리로 잡히고, 마이그레이션이
타깃에 그대로 올린다. 그 nginx가 곧 부하테스트 대상이 된다.

## RUN_ID — 리소스 이름 접미사

honeybee(소스그룹)·cicada(워크플로우) 등은 이름에 **UNIQUE 제약**이 있다. 같은 이름으로 다시 등록하면 거부되고,
에러 모달이 열린 채 남아 뒤 조작을 가린다. 그래서 실행마다 접미사를 붙인다 — `e2e-nano-source-842461` 처럼.

`RUN_ID` 는 **`globalSetup` 이 실행 시작 시 한 번 정해** 환경변수로 심고, 모든 worker 프로세스가 그 값을 물려받는다.

> ⚠️ seed·functional·scenario 는 **서로 다른 worker 프로세스**에서 돈다.
> 프로세스마다 따로 만들면 seed가 만든 `...-123456` 을 functional 이 `...-789012` 로 찾다가 못 찾는다.
> 예전에 그렇게 깨졌다. 직접 `E2E_RUN_ID` 를 넣을 일은 거의 없다.

## `.env` 를 쓰고 싶다면

Playwright 는 `.env` 를 자동으로 읽지 않는다. 셸에서 export 하거나 앞에 붙인다.

```bash
set -a; source .env.e2e; set +a
npx playwright test --config=tests/e2e/playwright.config.ts
```

> 🔒 **개인키·비밀번호는 절대 커밋하지 않는다.** `.env.e2e` 는 `.gitignore` 에 둔다.
